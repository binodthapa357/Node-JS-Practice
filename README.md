# NODE with EXPRESS API

## Node.js

- Node.js is a JavaScript run-time environment.
- Runtime: A program that runs another program.
- With the help of Node.js we can run JavaScript in local machine.
- Build on C++.
- Powered by Google Chrome v8 engine.
- Used to build API, real-time apps, micro-services.

## Node.js Architecture

- Single Threaded
- Non-blocking I/O Operations
- Event Loop

## Express.js
- It is Node.js API/backend framework.
- Used to build API (Application Program Interface)
- It simplifies http module of node.js
- Fast, unopinionated, minimalist web framework for Node.js
